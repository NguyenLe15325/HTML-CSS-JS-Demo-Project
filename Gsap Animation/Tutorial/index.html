1. Basic Tween - gsap.to()

Most common method. Animates FROM current state TO specified values.
gsap.to("#box1", { x: 300, duration: 1, ease: "power2.out" });
Box 1

2. From Animation - gsap.from()

Animates FROM specified values TO current state. Great for entrance animations.
gsap.from("#box2", { opacity: 0, y: -100, duration: 1, ease: "bounce.out" });
Box 2

3. Multiple Properties & Easing

Animate multiple properties simultaneously with various easing functions.
gsap.to("#box3", { x: 250, rotation: 360, scale: 1.5, duration: 2, ease: "bounce.out" });
Box 3

4. Timeline - Sequenced Animations

Create complex sequences with precise timing control.
const tl = gsap.timeline(); tl.to(".timeline-box", { x: 200, stagger: 0.2 });

6. ScrollTrigger - Scroll-Based Animation

Trigger animations based on scroll position. Scroll in the container!

7. Repeat & Yoyo

Create looping animations that play forward and backward.
gsap.to("#box7", { x: 200, duration: 1, repeat: -1, // -1 = infinite yoyo: true, ease: "power1.inOut" });
Loop

8. Instant Set - gsap.set()

Set properties instantly without animation. Useful for initial states.
gsap.set("#box8", { x: 200, backgroundColor: "#ff6b6b", scale: 1.2 });
Set Props

9. Callbacks - Lifecycle Hooks

Execute functions at different stages: onStart, onUpdate, onComplete.
gsap.to("#box9", { x: 200, onStart: () => console.log("Started"), onUpdate: () => console.log("Updating"), onComplete: () => console.log("Done!") });
Callbacks

10. Delay & Duration Control

Control animation timing with delay and duration parameters.
gsap.to("#box10", { x: 200, duration: 2, delay: 0.5, ease: "elastic.out(1, 0.3)" });
Delayed

11. FromTo - Full Control

Define both starting and ending values for complete control.
gsap.fromTo("#box11", { x: -100, opacity: 0 }, { x: 200, opacity: 1, duration: 1 } );
FromTo

12. Text Animation - Split Characters

Animate text character by character for engaging effects.
const chars = text.split(''); gsap.from(chars, { opacity: 0, y: -50, stagger: 0.05 });

üí° Code Organization Example

Here's how to structure your animations in animations.js:

// animations.js // Utility function for fade in function fadeIn(element) { return gsap.from(element, { opacity: 0, y: 20, duration: 0.6, ease: "power2.out" }); } // Utility function for slide in function slideIn(element, direction = "left") { const xValue = direction === "left" ? -100 : 100; return gsap.from(element, { x: xValue, opacity: 0, duration: 0.8, ease: "power3.out" }); } // Complex timeline animation function heroAnimation() { const tl = gsap.timeline(); tl.from(".hero-title", { y: -50, opacity: 0 }) .from(".hero-subtitle", { y: 20, opacity: 0 }) .from(".hero-button", { scale: 0, opacity: 0 }); return tl; } // Initialize on page load document.addEventListener('DOMContentLoaded', () => { fadeIn('.content'); heroAnimation(); });
